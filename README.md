# Suma-de-numeros-java
Programa para el parcial numero 1 Uni-Remington

## Estructura del Proyecto

Este proyecto contiene tres archivos Java:

1. **ISumaNumeros.java** - Interfaz con las firmas de métodos
2. **SumaNumeros.java** - Clase que implementa la interfaz
3. **Main.java** - Clase principal con el punto de entrada

## Cómo Compilar

```bash
javac ISumaNumeros.java SumaNumeros.java Main.java
```

## Cómo Ejecutar

```bash
java Main
```

## Funcionalidad

El programa permite:
- Sumar dos números enteros
- Salir del programa

El ciclo principal utiliza una variable `RUNNING` que es controlada por el método `salir()` de la interfaz.
